## 🚀 Features

- 🔋 Animated battery fill with shine sweep
- 🎨 Color shifts based on level — green → orange → red
- 💡 CSS glow effects & grid background
- 🎚️ Interactive slider to adjust battery level
- 📱 Responsive & mobile-friendly
- ⚡ Zero dependencies — plain HTML/CSS/JS

## 🛠️ How to Run

1. Clone this repo
